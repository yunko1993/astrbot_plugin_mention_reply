# AstrBot 插件：群友专属回复 (Mention Reply)

🎭 让群友拥有自己的“嘴替”！当有人在群里 @某位群友 时，机器人会自动代替该群友说出他预设的台词。适合小群互动、整活、挂机回复等场景。

## ✨ 功能特点
- **无感触发**：无需 @机器人，只要 @对应群友即可触发。
- **权限控制**：用户只能设置自己的台词，管理员可管理所有人。
- **全局开关**：管理员可随时开启/关闭功能，避免打扰。
- **防刷屏**：多人同时被 @ 时，合并为一条消息回复。
- **内置帮助**：输入 `/嘴替帮助` 即可查看所有指令，无需查阅文档。
- **持久化存储**：配置自动保存，重启不丢失。

## 🚀 安装方法

### 通过 AstrBot WebUI 安装 (推荐)
1. 打开 AstrBot 仪表盘。
2. 进入 **插件管理** -> **从 GitHub 安装**。
3. 输入本仓库的 URL。
4. 点击安装并启用。

### 手动安装
将本仓库克隆到 AstrBot 的插件目录 (`data/plugins/`)，然后重载插件。

## 📖 使用指令

在群聊中输入以下指令（无需 @机器人）：

| 指令 | 说明 | 权限 |
| :--- | :--- | :--- |
| `/嘴替帮助` 或 `/reply_help` | **查看帮助信息和所有指令用法** | 所有人 |
| `/setreply @某人 台词` | 设置某人的专属回复台词 | 本人或管理员 |
| `/delreply @某人` | 删除某人的专属回复台词 | 本人或管理员 |
| `/switch_mention` | 开启/关闭插件功能 | 仅管理员 |
| `/listreply` | 查看当前已设置台词的群友列表 | 所有人 |

## 💡 使用示例

1. **不知道怎么玩？**
   > 用户: `/嘴替帮助`
   > 机器人: 发送详细的使用指南...

2. **设置台词**
   > 用户A: `/setreply @老王 今天天气真好，适合摸鱼！`
   > 机器人: `✅ 已设置 [老王] 的专属回复...`

3. **触发回复**
   > 用户B: `@老王 你怎么不说话？`
   > 机器人: `🎭 老王 说：今天天气真好，适合摸鱼！`

## ⚙️ 配置文件
插件会自动在同目录下生成 `replies.json` 和 `status.json`，无需手动编辑。
