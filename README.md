# AstrBot 插件：群友嘴替助手 (Mention Reply)

🎭 让群友拥有自己的“嘴替”！当有人在群里 @某位群友 时，机器人会自动代替该群友说出他预设的台词，并**完美拦截大模型**，防止机器人自言自语。适合小群互动、整活、挂机自动回复等场景。

## ✨ 功能特点
- **无感触发**：无需 @机器人，只要 @对应群友即可触发代答。
- **精准提取**：采用纯原生字符串物理分割，完美兼容含有空格、特殊符号、多行文本的台词。
- **权限控制**：仅指定管理员（白名单）可设置和删除台词，防止群员互相恶搞。
- **全局开关**：管理员可随时一键开启/关闭插件功能。
- **完美兼容**：适配 AstrBot v4.16+ 官方全局监听器 (`EventMessageType.ALL`)，拦截底层消息，不抛异常。
- **持久化存储**：配置自动保存在插件目录下的 JSON 文件中，重启不丢失。

## 🚀 安装方法

### 通过 AstrBot WebUI 安装 (推荐)
1. 打开 AstrBot 仪表盘。
2. 进入 **插件市场** -> **通过 GitHub URL 安装**。
3. 输入本仓库的 URL：`https://github.com/yunko1993/astrbot_plugin_mention_reply`
4. 点击安装并重启 AstrBot。

### 手动安装
将本仓库克隆到 AstrBot 的插件目录 (`addons/plugins/` 或 `data/plugins/`)，并在 WebUI 启用后重启框架。

## 📖 使用指令

在群聊中输入以下指令（无需 @机器人）：

| 指令 | 说明 | 权限要求 |
| :--- | :--- | :--- |
| `/嘴替帮助` | **查看控制台和所有指令用法** | 所有人 |
| `/setreply @某人 台词` | 设置某人的专属回复台词 | 仅管理员 |
| `/delreply @某人` | 删除某人的专属回复台词 | 仅管理员 |
| `/listreply` | 查看当前已设置台词的群友列表 | 所有人 |
| `/toggle` | 开启/关闭插件全局代答功能 | 仅管理员 |

> ⚠️ **注意**：使用 `/setreply` 时，请确保 `@用户` 和后面的 `台词` 之间至少有一个空格。

## 💡 使用示例

1. **设置台词**
   > 用户A: `/setreply @老王 他现在不在，有事烧纸。`
   > 机器人: `✅ 设置成功！有人 @老王 时，我会代答：他现在不在，有事烧纸。`

2. **触发回复**
   > 用户B: `@老王 你怎么不说话？`
   > 机器人: `他现在不在，有事烧纸。` (大模型被成功拦截，不会额外回复)

3. **开关插件**
   > 用户A: `/toggle`
   > 机器人: `✅ 嘴替功能已关闭。`

## ⚙️ 配置文件
首次运行或设置后，插件会自动在自身目录下生成 `mention_reply_config.json` 文件。
如果你需要**添加其他管理员**，可以直接打开该文件，在 `admin_qq` 列表中加入对应的 QQ 号并重启机器人。

```json
{
    "enabled": true,
    "admin_qq":[
        "1023902556"
    ],
    "replies": {
        "123456789": "这是预设的台词"
    }
}
