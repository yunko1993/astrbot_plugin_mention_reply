# AstrBot 插件：群友嘴替助手 (Mention Reply)

🎭 让群友拥有自己的“嘴替”！当有人在群里 @某位群友 时，机器人会自动代替该群友说出他预设的台词，并**完美拦截大模型**。

## ✨ 功能特点
- **无感触发**：无需 @机器人，只要 @对应群友即可触发代答。
- **精准提取**：物理分割消息内容，完美兼容含有空格、特殊符号、多行文本的台词。
- **权限控制**：仅指定管理员（白名单）可设置和删除台词。
- **全局开关**：管理员可随时一键开启/关闭插件功能。
- **完美兼容**：适配 AstrBot v4.16+ 官方全局监听逻辑。
- **持久化存储**：配置自动保存在插件目录下的 JSON 文件中，重启不丢失。

## 🚀 安装方法

### 通过 AstrBot WebUI 安装 (推荐)
1. 打开 AstrBot 仪表盘。
2. 进入 **插件市场** -> **通过 GitHub URL 安装**。
3. 输入本仓库的 URL：`https://github.com/yunko1993/astrbot_plugin_mention_reply`
4. 点击安装并重启 AstrBot。

## 📖 使用指令

在群聊中输入以下指令：

| 指令 | 说明 | 权限要求 |
| :--- | :--- | :--- |
| `/嘴替帮助` | 查看控制台和所有指令用法 | 所有人 |
| `/setreply @某人 台词` | 设置某人的专属回复台词 | 仅管理员 |
| `/delreply @某人` | 删除某人的专属回复台词 | 仅管理员 |
| `/listreply` | 查看当前已设置台词的列表 | 所有人 |
| `/toggle` | 开启/关闭插件全局代答功能 | 仅管理员 |

> ⚠️ 注意：使用 /setreply 时，@用户 和 台词 之间请保留一个空格。

## 💡 使用示例

1. **设置台词**
   > 用户A: `/setreply @老王 他现在不在，有事烧纸。`
   > 机器人: `✅ 设置成功！有人 @老王 时，我会代答：他现在不在，有事烧纸。`

2. **触发回复**
   > 用户B: `@老王 你怎么不说话？`
   > 机器人: `他现在不在，有事烧纸。`

## ⚙️ 配置文件
首次运行或设置后，插件会在自身目录下生成 `mention_reply_config.json`。如需添加管理员，请修改 `admin_qq` 列表。

```json
{
    "enabled": true,
    "admin_qq":[
        "1023902556"
    ],
    "replies": {
        "123456789": "这是预设的台词"
    }
}
```
